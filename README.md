# talib
wrapper for TA-Lib, provide intellisense in vscode and PyCharm

让TA-Lib支持智能感知

## Usage

```bash
pip install talib
```

```python
import numpy
import ulib
close = numpy.random.random(100)
ulib.SMA(close)
```
